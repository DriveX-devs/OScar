# ASN.1 generated code

This directory contains C code generated from ETSI ASN.1 specifications using
`asn1c` and `asn2c`.

The original ETSI `.asn` files are licensed under BSD-3-Clause unless otherwise stated.
These files are **not** included here.

The `asn1c` and `asn2c` compilers and their generated support/skeleton code are licensed under
BSD-2-Clause.

These files are included in OScar under their original permissive licenses and
are not relicensed under Apache-2.0.
